SISTEMA INFOCLIN

Esse sistema permite automatizar o processo de atendimento de uma clínica médica, fornecendo recursos que permitam a melhor execução e condução de diagnósticos, exames e agendamentos de consultas, possibilitando a integração gerencial da clínica.

Este sistema está sendo desenvolvido como parte da atividade integradora multidisciplinar de Projeto de Desenvolvimento de Sistemas Distribuídos dos alunos do 4º Período do curso de Análise e Desenvolvimento de Sistemas, semestre 2011.1.
